# ChatGPT Action Figure Generator

A modern web platform that helps users create and explore ChatGPT action figures through AI image generation.

## Features

### 1. Tutorial Section
- Step-by-step guide on creating ChatGPT action figures
- Clear instructions with visual aids
- Best practices and tips for optimal results

### 2. Prompt Library
- Curated collection of effective prompts
- Example images for each prompt
- One-click copy functionality
- Categories for different styles and themes

### 3. Gallery Showcase
- Display of celebrity ChatGPT action figures
- Interactive like/dislike voting system
- Responsive grid layout
- Social sharing capabilities
